genLidar

Generate a reading for the LIDAR sensor

Syntax

distLidar = genLidar(obj)

Input Variable

obj - an instance of the CreateRobot class

Output Variable

distLidar - Vector of doubles of length obj.numPtsLidar. Distances correpond to angles in the range [-angRangeLidar/2 angRangeLidar/2]. Remember that angles are defined positive counter-clockwise.
