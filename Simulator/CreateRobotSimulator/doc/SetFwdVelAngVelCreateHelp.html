SetFwdVelAngVelCreate

Controls the forward and angular velocity of the robot

Syntax

SetFwdVelAngVelCreate(obj,FwdVel,AngVel)

Input Variables

obj - an instance of the CreateRobot class
FwdVel - Double, forward velocity in the range [-.5 .5] (m/s)
AngVel - Double, angular velocity in the range [-2.5 2.5] (rad/s)
Note: the combination of forward and angular velocity should not cause each individual wheel exceed their velocity limits of [-0.5 0.5] (m/s)

Output Variable

The function SetFwdVelAngVelCreate produces no output.
